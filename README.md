# 58 同城租房信息爬虫

## 📋 项目简介
这是一个用于爬取 58 同城上海地区租房信息的 Python 脚本，可以自动收集房源数据并保存到 Excel 表格中。

## 🎯 功能特点
- 自动爬取 58 同城上海租房频道的房源信息
- 支持多页连续爬取（默认 1-70 页）
- **智能代理 IP 支持**：自动获取代理 IP，突破访问限制
- **自动重试机制**：失败自动换 IP 重试，最多 8 次
- **Cookie 自动获取**：自动调用浏览器获取登录态
- 提取以下关键信息：
  - 房屋标题
  - 价格（元/月）
  - 房屋类型和面积
  - 所在地点
  - 房源链接
- 自动保存数据到桌面 Excel 文件
- 支持数据追加模式（不会覆盖已有数据）
- 自动添加序号

## 🚀 使用方法

### 环境要求
- Python 3.x
- 需要安装的依赖包：
```bash
pip install pandas lxml requests openpyxl selenium
```

### 运行步骤
1. 确保已安装所有依赖包
2. 确保已下载 WebDriver（msedgedriver.exe）
3. 直接运行脚本：
```bash
python "简易 58city.py"
```
4. 程序会自动获取 Cookie 和代理 IP
5. 爬取完成后，数据会自动保存到桌面的 `上海租房信息表.xlsx` 文件中

## 🛠️ 常见问题

**Q: 为什么爬取不到数据？**
A: 可能是网络问题、页面结构变化或触发了反爬机制，可以尝试：
- 检查网络连接
- 降低爬取速度
- 程序会自动切换代理 IP 重试

**Q: 如何停止爬取？**
A: 按 Ctrl+C 可以中断程序运行

**Q: 数据保存在哪里？**
A: 默认保存在电脑桌面的 `上海租房信息表.xlsx` 文件中

**Q: 为什么自动打开浏览器？**
A: 首次运行需要获取 58 同城的 Cookie，用于绕过登录验证

**Q: 遇到验证码怎么办？**
A: 程序会自动使用代理 IP 尝试绕过，如果仍然遇到验证码，建议稍后再试

**Q: 如何手动获取 Cookie？**
A: 运行 `58 同城 cookie 自动化获取.py` 脚本，手动登录后会自动保存 Cookie
