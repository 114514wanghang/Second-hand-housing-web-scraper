# 58 同城租房信息爬虫

## 📋 项目简介
这是一个用于爬取 58 同城上海地区租房信息的 Python 脚本，可以自动收集房源数据并保存到 Excel 表格中。

## 🎯 功能特点
- 自动爬取 58 同城上海租房频道的房源信息
- 支持多页连续爬取（默认 1-70 页）
- 提取以下关键信息：
  - 房屋标题
  - 价格（元/月）
  - 房屋类型和面积
  - 所在地点
  - 房源链接
- 自动保存数据到桌面 Excel 文件
- 支持数据追加模式（不会覆盖已有数据）
- 自动添加序号

## 🚀 使用方法

### 环境要求
- Python 3.x
- 需要安装的依赖包：
```bash
pip install pandas lxml requests openpyxl
```

### 运行步骤
1. 确保已安装所有依赖包
2. 直接运行脚本：
```bash
python "简易 58city.py"
```
3. 爬取完成后，数据会自动保存到桌面的 `上海租房信息表.xlsx` 文件中
